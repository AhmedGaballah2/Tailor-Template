# Tailor  Template
A template for a tailor shop using HTML5, CSS3 and JS.

## About Tailor

This project is an example that was built along with the ITI Client-Side course showing how to create a landing page (template) using HTML, CSS and JS. Every part of this project is sample code which shows how to do the following

 Creating a static template using HTML, CSS.
 Adding some animations to the elements to give them some spirit.
 Coordinating colors and images with each other in a way that is convenient for the user.

If you want to check fast what is this website, you can play the `Video Project 1.mp4` video that shows a screen record of the website.

## Quick start
### How to download this template

You can use the `Download ZIP` button in the top-right ` Code` button.

## How to use this template

1. Opening the template
   If you have downloaded the zip file, unzip the folder, then open the `tailor.html` file.
   
3. Using the template
   ### HTML
     - Start with `tailor.html`, this will be your root page (main page) to change any element.
   ### CSS
     - Start with `tailor.css`, to change any style for any element.
   ### JavaScript
     - Start with `tailor.js`, to change any logical method in the page or any animations.
   ### Images
     - Start with any `img` inside the `images` folder, you can change any `img` you want with another one.
